# Генератор тестовых данных для CSV Median Calculator

Скрипт на Python для генерации тестовых CSV-файлов с биржевыми данными.

## Требования

- Python 3.6+
- Стандартная библиотека (csv, random, os, datetime)

## Использование

```bash
# Перейти в папку со скриптом
cd scripts

# Запустить генерацию
python3 generate_test_data.py
```

## Результат

### Скрипт создаст файлы в ../examples/input/:

- `btcusdt_level_2024.csv` — данные стакана (200 строк)
- `btcusdt_trade_2024.csv` — данные сделок (150 строк)

## Гибкая настройка

### Перед запуском можно изменить основные параметры:

```python
BASE_TIME = 1716810808593627  # начальный timestamp
PRICE_BASE = 68480.0          # базовая цена
OUTPUT_DIR = "../examples/input"  # папка для выходных файлов
```
